# proxifier

!> 支持网络协议之间的转换，用于打开不同网络协议的软件进行互联，试学一月不收任何费用

 未配置网络协议的情况下，打开telegram，连接是不成功的，此时打开proxifier协议代理软件，在`profile`中选择`proxy servers`
