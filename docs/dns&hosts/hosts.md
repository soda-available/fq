# 修改hosts查询维基百科

!> 早期，GFW屏蔽的网址不是很多且手段也仅限于污染，修改hosts也仅对当前配置所受污染的站点有效。


利用 https://www.ipaddress.com 查询维基百科相关ip，并找到`C:\Windows\System32\drivers\etc\host`，打开它
