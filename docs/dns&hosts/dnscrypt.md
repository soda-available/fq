# DNSCrypt打开论坛站点

!> 目前，多数SS分享站点或是VPN官网均以污染为主，因此可用它来打开绝大部分此类相关网址，并以此下载VPN或是配置代理从而连接互联网；但需注意遇到如谷歌等这类封IP的网站是打不开的。 <br><br>
多说一句，仅针对一些相对来说大众不很明白或是易混淆的相关概念进行此类提示，若还需较为深入地了解相关的原理细节等，还请自行翻看科普与补充的相关章节（ps：论坛现已封ip，2019）
